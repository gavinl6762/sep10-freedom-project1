## Content

<p>In SEP10 we had to do a big project and currently we're choosing a tool for the project. There are several tools to be used, ranging from mild to medium to spicy and finally fire, mild being the easiest to understand and fire to being more difficult to understand. The tool I chose was Aframe, Aframe is a web framework used for creating 3D virtual reality experiences. A way i tinkered with Aframe was creating like a small plain area with a blue block in the middle. First, I basically copied the script into my IDE which is a given code in the Aframe introduction website, it came with a code that worked in a 3D VR world. Then i added the elements like `a-sky` which does things with the sky, `a-plane` which affects the base/ground, and `a-box` which affects the cube element. With the a-plane I made the height 100 making it stretch out longer and making the color green so it looks more like the ground. For the sky, i made it light blue so it'll look like a normal life. Then i added a box in the middle of the a-plane by setting the position at 0, 1, -3. 

## Skills

<p> Some skill i developed when tinkering the tool was knowing more about how to use Aframe and what each element do. This'll be important in the future because it can help me with designing 3D things especially in the Freedom Project. </p>
